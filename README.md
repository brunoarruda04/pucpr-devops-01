# 🚀 Ruby Tasks API (DevOps Project)

Projeto desenvolvido para as atividades formativas da disciplina de DevOps.

Aplicação simples em Ruby usando Sinatra, com pipeline completo de CI/CD, Docker e versionamento automático com semantic-release.

---

## 📦 Tecnologias utilizadas

- Ruby (Sinatra)
- Docker
- GitHub Actions (CI/CD)
- RuboCop (lint)
- Husky + Commitlint (padronização de commits)
- Semantic Release (versionamento automático)

---

## ⚙️ Como rodar o projeto

### 🔧 Pré-requisitos

- Ruby (>= 3.3 recomendado)
- Node.js (>= 20)
- Docker (opcional)

---

### 🚀 Setup completo

```bash
make setup
```
